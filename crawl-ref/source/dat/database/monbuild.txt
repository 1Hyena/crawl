# monbuild.txt: generate monster specs (for use with mons_list)
#
# Monster specs take the same format as MONS and KMONS in vaults:
# https://crawl.develz.org/wiki/doku.php?id=dcss:help:maps:syntax:monster_info
#
# Used by Demigod mechanic to generate minions for each god. It looks for a key
# with the format "{god name} minion {level}". Minions will be used from the
# highest available definition for their level (HD will be automatically adjusted).
#
# e.g: if an Ashenzari minion of level 14 is requested, we will fall back to
# @Ashenzari minion 9@ (but adjust the HD to 14).
#
# The minion level is decided by player XL, plus up to 3-4 extra levels based on
# piety and god wrath.
#
# -- mumra
#
